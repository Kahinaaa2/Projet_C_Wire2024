
. Implémentation
                                                                                                      .1 Développement du script Shell
                                                                                                      
                                                                                                          Étapes :
                                                                                                              Implémenter les options (type de station, type de consommateur, filtre par centrale, etc.).
                                                                                                              Ajouter les vérifications d’entrées (validation des paramètres).
                                                                                                              Vérifier la présence du programme C, lancer la compilation si nécessaire.
                                                                                                              Filtrer les données CSV avant de les transmettre au programme C.
                                                                                                          Livrable : Script c-wire.sh avec des commentaires détaillés.

                                                      .2 Développement du programme C
                                                      
                                                          Étapes :
                                                              Implémenter la lecture des données filtrées.
                                                              Construire l’AVL et y insérer les données.
                                                              Implémenter les fonctions pour le calcul des consommations.
                                                              Gérer les erreurs et assurer la robustesse du code.
                                                              Générer des fichiers de sortie au format spécifié.
                                                          Livrable :
                                                              Code en C.
                                                              Fichier Makefile pour la compilation.
